# 📂 Estruturas Implementadas

## 🌳 Árvore Binária de Busca

Implementação de uma **Binary Search Tree (BST)** com operações básicas.

**Funcionalidades:**

* Inserção de elementos
* Busca de valores
* Percurso da árvore
* Estrutura hierárquica de dados

**Como executar:**

```bash
python arvore_binaria.py
```

---

## 🌳 Árvore AVL

Implementação de uma **árvore AVL**, uma árvore binária de busca auto-balanceada.

**Funcionalidades:**

* Inserção balanceada
* Rotações (esquerda e direita)
* Cálculo de fator de balanceamento
* Percurso da árvore

**Como executar:**

```bash
python arvore_avl.py
```

---

## 🌳 Árvore Red-Black

Implementação de uma **árvore Rubro-Negra**, uma árvore binária balanceada utilizada em diversas estruturas de dados modernas.

**Funcionalidades:**

* Inserção com balanceamento automático
* Rotações na árvore
* Regras de cores (vermelho e preto)
* Estrutura eficiente para busca

**Como executar:**

```bash
python arvore_red_black.py
```

---

## 📥 Fila (Queue)

Estrutura de dados que segue o princípio **FIFO (First In, First Out)**.

**Funcionalidades:**

* Inserção de elementos
* Remoção de elementos
* Verificação de fila cheia ou vazia
* Controle de tamanho

**Como executar:**

```bash
python fila.py
```

---

## 📚 Pilha (Stack)

Estrutura baseada no princípio **LIFO (Last In, First Out)**.

**Funcionalidades:**

* Empilhar elementos
* Desempilhar elementos
* Verificação do topo da pilha
* Controle de tamanho

**Como executar:**

```bash
python pilha_stack.py
```

---

## 📋 Lista Ordenada

Estrutura que mantém os elementos **automaticamente organizados em ordem crescente**.

**Funcionalidades:**

* Inserção ordenada
* Busca de elementos
* Exibição da lista

**Como executar:**

```bash
python lista_ordenada.py
```

---

## #️⃣ Tabela Hash

Implementação de **Hash Table** com tratamento de colisões.

**Funcionalidades:**

* Inserção de chave e valor
* Busca de elementos
* Acesso estilo dicionário
* Exibição da tabela

**Como executar:**

```bash
python tabela_hash.py
```

---

## 🔺 Heap (Min Heap)

Implementação da estrutura **Heap**, utilizada em filas de prioridade e algoritmos eficientes.

**Funcionalidades:**

* Inserção de elementos
* Remoção do menor elemento
* Reorganização automática da estrutura (heapify)
* Visualização da heap

**Como executar:**

```bash
python heap.py
```

---

## 🌐 Grafos

Implementação de grafos utilizando **lista de adjacência**.

**Funcionalidades:**

* Adicionar vértices
* Adicionar arestas
* Visualizar conexões do grafo
* Algoritmos de busca BFS e DFS

**Como executar:**

```bash
python grafo.py
```
